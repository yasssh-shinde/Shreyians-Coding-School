üéØ CSS Selectors

Universal (*)
Selects all elements
* { margin: 0; padding: 0; }
Element (p, div, h1)
Selects all elements of that type
p { color: blue; }
Class (.classname)
Selects elements with specific class
.highlight { background: yellow; }
ID (#idname)
Selects element with specific ID (unique)
#header { background: navy; }
Attribute [attr]
Selects elements with specific attribute
input[type="text"] { border: 1px solid gray; }
Descendant (parent child)
Selects elements inside another element
div p { color: red; }
Child (parent > child)
Selects direct children only
ul > li { list-style: none; }
Adjacent Sibling (+)
Selects next sibling element
h1 + p { margin-top: 0; }
General Sibling (~)
Selects all following sibling elements
h1 ~ p { color: gray; }
Pseudo-class (:hover, :focus)
Selects elements in specific state
a:hover { color: red; }
Pseudo-element (::before, ::after)
Styles specific parts of elements
p::first-line { font-weight: bold; }
Multiple Selector (,)
Applies styles to multiple selectors
h1, h2, h3 { color: blue; }

üì± Responsive Design

@media Query
Applies styles based on conditions
@media (max-width: 768px) { .container { width: 100%; } }
max-width & min-width
Sets maximum and minimum element width
max-width: 1200px; min-width: 320px;
max-height & min-height
Sets maximum and minimum element height
max-height: 500px; min-height: 200px;
aspect-ratio
Maintains aspect ratio of element
aspect-ratio: 16 / 9; /* maintain 16:9 ratio */
viewport units
Sizes relative to viewport
width: 100vw; height: 50vh; /* vw, vh, vmin, vmax */
Mobile-first approach
Base styles for mobile, add breakpoints
/* Mobile */ .container { width: 100%; } /* Tablet */ @media (min-width: 768px) { ... } /* Desktop */ @media (min-width: 1024px) { ... }
Common Breakpoints
Standard screen sizes for responsive design
Standard sizes:
  • 320px - Small phones
  • 768px - Tablets
  • 1024px - Desktops
  • 1440px - Large screens
rem & em units
Relative sizing units
/* em = relative to parent element */ font-size: 1.5em; /* rem = relative to root element */ font-size: 1.5rem;
clamp()
Responsive sizing function
font-size: clamp(1rem, 5vw, 2rem);
calc()
Performs calculations in CSS
width: calc(100% - 20px);

üé≠ Pseudo-classes & Pseudo-elements

:hover
Styles element on mouse hover

Styles element when focused
input:focus { border-color: blue; outline: none; }

Styles element when clicked
button:active { transform: scale(0.95); }
:visited
Styles visited links
a:visited { color: purple; }
:first-child
Selects first child element
li:first-child { font-weight: bold; }
:last-child
Selects last child element
li:last-child { border-bottom: none; }
:nth-child(n)
Selects element by position
li:nth-child(2n) { background: gray; } /* even items */
:not(selector)
Selects elements NOT matching selector
p:not(.special) { color: black; }
::before
Inserts content before element
p::before { content: "‚Üí "; color: blue; }
::after
Inserts content after element
p::after { content: " ‚Üê"; color: blue; }
::first-line
Styles first line of text

::first-letter
Styles first character
p::first-letter { font-size: 2em; }
::selection
Styles selected text
::selection { background: yellow; color: black; }
:disabled & :enabled
Styles disabled/enabled form elements
input:disabled { opacity: 0.5; cursor: not-allowed; }
:checked
